3D Printed Stair Planter
Overview

This project is a 3D printed plant pot system with a stair-style design and simple passive irrigation. It is designed to be easy to print and use without electronics.

Features
Stair-step layout
Gravity-based irrigation
No electronics required
Easy to print
Modular design
Design

Each pot includes a drainage hole. Water added to the top level flows down to lower levels. The design helps distribute water and reduce overwatering.

Setup
Print all parts
Arrange in stair formation
Add soil and plants
Water from the top
